# Simple Property Manager

Simple site meant to be hosted on a local device (no online connection needed)
Built on the following stack:

Frontend: React
Backend: Node.js
Database: SQLite

Features:
View list of properties
Tenant Name
Rent received toggle
Notes

## Startup

To initialize, you need 2 terminals open at the same time

### Terminal 1

cd backend
node server.js

### Terminal 2

cd frontend
npm run dev
